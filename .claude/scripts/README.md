# Scripts

Automation scripts for component management.

## Available Scripts

### validate-components.sh
Validates component structure and manifest integrity.

### sync-to-local.sh
Syncs components from repository to local installation.

## Usage

```bash
# Run validation
./scripts/validate-components.sh

# Sync to local Claude Code installation
./scripts/sync-to-local.sh
```

## CI/CD Integration

Scripts are used by GitHub Actions workflows in `.github/workflows/`.
